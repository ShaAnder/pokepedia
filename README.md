<h2 class="header2">Description</h2>
<p class="text-content">
A beginner level project showcasing the ability to create a webpage based on no information except an example. The example we're using is the US pokedex webpage, which unlike the current application is built in JS. I tasked myself with the job of taking that webpage and rebuilding it in react, with the methodology of SPA, currently the application has basic and advanced sorting features based on the pokemon. My next goal is to make it responsive and create a modal window that will allow the user to view pokemon details.
